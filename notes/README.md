# Notes

Scratch space, brain dumps, meeting notes. No structure enforced — keep it scannable.

## Conventions

- File by date: `YYYY-MM-DD-<slug>.md`
- Top of each file: one-line summary
- No status fields, no templates — this is "stuff I want to remember"
- Anything that becomes a real decision moves to `adrs/`
- Anything that becomes a real plan moves to `plans/`
